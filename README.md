Electric Vehicle (EV) Data Analysis - MIS 311
1. Project Overview
This project focuses on exploratory data analysis (EDA) of electric vehicle (EV) registrations across different cities. The dataset includes details such as vehicle manufacturers, models, model years, and registration locations.
Purpose
The purpose of this analysis is to examine trends and patterns in electric vehicle registrations, identify the most popular manufacturers, understand how EV adoption has changed over time, and discover which cities have the highest number of registrations.

2. Data Cleaning & Preparation
The raw dataset was cleaned and processed using Microsoft Excel to improve accuracy and reliability.
Cleaning Process
-	Missing values were identified and rows with incomplete information were removed.
-	Duplicate records were removed using Excel’s “Remove Duplicates” function.
-	The dataset was checked to ensure consistency and accuracy before analysis.

3. Descriptive Statistics & Key Insights
Insight 1: Most Common Manufacturers
Finding: Tesla has the highest number of vehicles in the dataset, with 74 out of 188 observations. It exceeds other manufacturers such as Nissan and Kia by a significant margin.
Conclusion: This suggests that Tesla has a strong market presence and high consumer demand in the electric vehicle industry.

Insight 2: Model Year Distribution
Finding: The number of electric vehicles increased significantly between 2018 and 2020, reaching the highest level in 2020. Vehicle registrations gradually increased again after 2021.
Conclusion: This trend indicates growing consumer interest in newer electric vehicle technologies and sustainable transportation.
